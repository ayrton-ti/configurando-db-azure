# Configurando Banco de Dados na Azure

## 📌 Introdução
A computação em nuvem vem transformando a forma como aplicações e serviços são desenvolvidos e disponibilizados, oferecendo escalabilidade, segurança e alta disponibilidade. Nesse contexto, a Microsoft Azure se destaca como uma das principais plataformas de cloud computing.

Este repositório foi desenvolvido com o objetivo de documentar a experiência prática na configuração de uma instância de banco de dados na Azure, reunindo conceitos, aprendizados e boas práticas adquiridas ao longo do laboratório.

---

## 🎯 Objetivos
- Aplicar na prática conceitos fundamentais de computação em nuvem;
- Realizar a configuração de uma instância de banco de dados na Azure;
- Documentar processos técnicos de forma clara e organizada;
- Utilizar o GitHub como ferramenta de compartilhamento de conhecimento.

---

## 🚀 Etapas realizadas

### 🔹 1. Criação e acesso à conta Azure
Inicialmente, foi realizado o acesso ao portal da Microsoft Azure, explorando a interface e compreendendo a organização dos recursos dentro da plataforma.

### 🔹 2. Criação da instância de banco de dados
Foi realizada a criação de uma instância de banco de dados utilizando os serviços disponíveis na Azure, com atenção aos seguintes pontos:
- Escolha do tipo de banco (Azure SQL);
- Definição de nome do servidor e banco;
- Configuração de login e senha de administrador;
- Seleção da região e recursos de desempenho.

### 🔹 3. Configuração de rede e segurança
Durante o processo, foram configuradas regras básicas de acesso, permitindo a conexão ao banco de dados de forma segura, respeitando boas práticas de segurança.

### 🔹 4. Revisão e validação da configuração
Após a criação, foram revisadas todas as configurações para garantir que o ambiente estivesse funcional e pronto para uso.

---

## 🧠 Aprendizados

Durante a realização deste laboratório, foi possível consolidar diversos conceitos importantes, como:

- Entendimento da estrutura da Microsoft Azure;
- Processo de criação e gerenciamento de recursos na nuvem;
- Configuração de bancos de dados em ambiente cloud;
- Importância da segurança e controle de acesso;
- Organização e documentação de processos técnicos.

Além disso, foi possível perceber na prática como a computação em nuvem facilita a criação de ambientes escaláveis e acessíveis, reduzindo a complexidade de infraestrutura local.

---

## 💡 Considerações finais

A realização deste laboratório contribuiu significativamente para o desenvolvimento prático em cloud computing, reforçando a importância da documentação e da organização das informações técnicas.

Este repositório serve como material de apoio para estudos futuros e também como base para implementação de soluções em ambientes reais utilizando a plataforma Microsoft Azure.
